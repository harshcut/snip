### Introduction

Capture screenshots, detect React elements, and send visual context with source code references directly to AI.

Snip is a companion devtool that helps you send visual context and source code references from your browser to Cursor IDE. The extension works in the background to detect React components and resolve their source file paths. When you capture a region of the page, the side panel will show an overview of the detected components, which includes their source files and line numbers. You can then write your prompt and send it to Cursor along with the screenshot and source paths via a deeplink.
